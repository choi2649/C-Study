주석사용방법
	- 주석이란? 컴파일러가 코드를 무시하게 만드는 것이다.
	아래 코드중 //가 되어있는것은 무시된다.
		int x = 2;
		//int x = 2;
	-또 다른방법(영역주석)
	/*
		int x = 2;
		int x = 2;
	*/
	- 프로그래머들이 메모할때 사용한다.
	- 자세한 설명을 달아주는것이 좋다. 
	- 코드만보고 알수있는정보는 적을필요가 없다.
