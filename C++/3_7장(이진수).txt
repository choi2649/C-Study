이진수
이진수는 0과1로만 표현한다.
컴퓨터는 전압이 높고 낮고로 표현하게된다.

1Byte는 8bit이다.
아래는 8bit를 이진수를 표현한것이다.
0101 1110

2진수 0101 1110를 10진수로 변경

2^7*0 + 2^6*1 + 2^5*0 + 2^4*1 +
2^3*1 + 2^2*1 + 2^1*1 + 2^0*0
으로 계산할 수 있다.더 쉽게 생각하면
작은 자리부터 1 2 4 8 16 32 64 128식으로 생각하면 된다.

10진수 148을 2진수로 변경
148 / 2 = 74 나머지 0
74  / 2 = 37 나머지 0
37  / 2 = 18 나머지 1
18  / 2 = 9  나머지 0
9   / 2 = 4  나머지 1
4   / 2 = 2  나머지 0
2   / 2 = 1  나머지 0
1   / 2 = 0  나머지 1

10010100 이 되는걸 볼 수 있다.
